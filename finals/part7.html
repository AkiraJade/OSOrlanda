PART 7: WEB SERVER

IIS 7 Web Server Deployment for TUP-Taguig University

This video demonstrates the comprehensive installation and configuration of Internet Information Services (IIS) 7.0 on Windows Server 2008 to host the TUP-Taguig university website and departmental sub-sites. The web server provides a secure and scalable platform for hosting web applications, content management, and online services for the entire university community.

Web Server Implementation:
• Main University Website: Simple, professional website showcasing TUP-Taguig with information about programs, departments, and services
• Departmental Sub-sites: Dedicated websites for each academic department and administrative office created as subdirectories under wwwroot
• Secure Configuration: HTTPS/SSL support, proper authentication, and security hardening for production deployment

Departmental Website Structure:
• Basic Art and Sciences Department website
• Electrical and Allied Department website
• Mechanical and Allied Department website
• Civil and Allied Department website
• Administrative offices sub-sites (Registrar, Director's Office, HR, Finance, etc.)

Technical Features:
• Support for ASP.NET, PHP, and static HTML content
• FTP services for content management and updates
• Application pools for isolation and security
• Performance optimization and caching

The implementation provides a centralized web platform for all TUP-Taguig online services and departmental presence.
