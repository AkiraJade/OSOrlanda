PART 8: GROUP POLICIES

Comprehensive Group Policy Implementation for TUP-Taguig Network Security

This video demonstrates the extensive configuration of Group Policy Objects (GPOs) to enforce security policies, user restrictions, and standardized desktop environments across all TUP-Taguig workstations. Group Policies provide centralized management of user and computer settings, ensuring consistent security and operational standards throughout the university network.

Desktop Customization:
• Department-specific desktop wallpapers for visual identification
• Customized Internet Explorer title bar showing domain name (tupt.edu.ph)

Security Restrictions:
• Control Panel Access: Prohibited for users, accessible only by administrators
• Command Prompt: Disabled for users to prevent unauthorized system access
• Registry Editing: Blocked for users to prevent system modifications
• Removable Storage: All USB and external storage devices denied for data security
• Software Installation: Users prohibited from installing applications
• Guest Account: Disabled across all workstations

Password Policies:
• Minimum password length: 12 characters
• Maximum password age: 42 days (enhanced from 30 days)
• LAN Manager hash disabled on password changes
• Account lockout: 3 invalid attempts triggers 30-minute lockout

Start Menu Restrictions:
• Windows Update: Disabled for users, enabled for admins
• Network Connections: Removed from user start menu
• Help Menu: Removed from user start menu
• Run Command: Removed from user start menu
• My Pictures/Music/Network Places: Removed from user start menu
• Logoff: Added to start menu for all users
• Task Manager: Disabled for users via Ctrl+Alt+Del

Administrative Controls:
• Directory service access auditing enabled and restricted to IT admins
• Automatic updates configured without user restarts
• Network connection properties restricted for users

The implementation ensures a secure, standardized, and controlled computing environment across all TUP-Taguig workstations.
