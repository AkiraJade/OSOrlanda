PART 4: FILE SHARING

Network File Sharing Configuration for TUP-Taguig Departments

This video demonstrates the comprehensive setup of network file sharing services on Windows Server 2008 for all TUP-Taguig departments and offices. The configuration ensures secure, organized, and efficient file management across the university network while maintaining proper access controls and storage quotas.

File Sharing Structure:
• Department Folders: Each department/section has a dedicated public folder with read/write access, restricted to authorized department users only
• University Public Folder: Central repository accessible to all domain users for cross-department collaboration
• Installer Folder: Application repository with administrator-only access for software distribution

Access Controls & Restrictions:
• File Type Restrictions: Most folders allow only office documents (.ppt, .doc, .docx, .xls, .xlsx, .pdf)
• IT Department Exception: Information Technology folder allows all file types except executables for development purposes
• Storage Quotas: Each folder limited to 1GB to ensure fair resource allocation
• Mapped Drives: Shared folders automatically mapped for each client workstation based on user department

The implementation ensures data security while facilitating efficient collaboration within and between departments.
