PART 1: DOMAIN CONTROLLER

Domain Name: tupt.edu.ph

This video demonstrates the installation and configuration of Windows Server 2008 as a Domain Controller for the TUP-Taguig network infrastructure. The domain controller serves as the central authentication and authorization hub for all users and computers in the university network. Key configuration steps include Active Directory Domain Services installation, domain creation, and initial domain controller setup to establish the foundation for user management, group policies, and network security across all academic and administrative departments.
