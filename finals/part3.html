PART 3: DHCP SERVER

DHCP Configuration for TUP-Taguig Network

This video demonstrates the installation and configuration of DHCP (Dynamic Host Configuration Protocol) server role on Windows Server 2008 for the TUP-Taguig network infrastructure. The DHCP service automatically assigns IP addresses and network configuration parameters to client devices, eliminating manual IP assignment and reducing network administration overhead.

Key DHCP Configuration Parameters:
• IP Address Blocks: 172.31.0.0/16 and 10.0.0.0/8 for comprehensive network coverage
• Router IP: Configured as valid gateway address for proper network routing
• DNS Servers: Primary DNS set to DHCP server IP, with Google DNS (8.8.8.8) and OpenDNS (208.67.222.222) as secondary servers for redundancy
• IP Lease Duration: 8 hours to balance between efficiency and IP address management
• IP Address Exclusions: First 30 and last 30 IP addresses reserved for network infrastructure and servers

The configuration ensures reliable network connectivity for all academic and administrative workstations while maintaining proper network segmentation and security.
